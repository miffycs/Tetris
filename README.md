# Tetris

:video_game: A Java implementation of *Tetris*.  
Made with: `java.awt` & `javax.swing`

## Demo
* Download `Tetris.jar` to play.

</br>

## Controls
| Key | Function |
| :---: | --- |
| `←` | Move left |
| `→` | Move right |
| `↓` | Move down |
| `↑` | Rotate piece |
| `space` | Drop piece |
